# Security Policy

## Supported Versions

| Application Version | API Version | Supported          |
|---------------------|-------------|--------------------|
| 5                   | 3           | :white_check_mark: |
| latest              | 3           | :white_check_mark: |
| < 5                 | any         | :x:                |

## Reporting a Vulnerability

Please report vulnerabilities as an issue in github. It will be put on top of backlog.
